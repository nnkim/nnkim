Enter Score and Get Grade

Code that return grades when enter scores.


Make a program that inputs a student's score and returns grade.

Condition :
1. The input of the function is received in integer
2. The return of the function is returned as a string
3. The score entered is calculated according to the score range

Grade Range:
Scores from 90 to 100 are A
Scores below 90 to 80 and above are B
Scores below 80 to 70 and above are C
Scores below 70 to 60 and above are D
Scores below 60 are F
